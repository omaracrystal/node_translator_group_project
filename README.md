# Translator-Node Group Project

## Description

A fun translator tool that uses helps users learn new languages and take challenges and track their progress.

## Technologies
- HTML5 & CSS3
- JavaScript / jQuery
- Bing API
- NodeJs / Express
- Swig
- Gulp
- MongoDB
- User Authentication
 -Bootstrap
